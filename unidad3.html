Unidad 3: Fundamentos de Programación Web

Introducción

La programación web es una disciplina fundamental en el desarrollo de aplicaciones interactivas y dinámicas en la web. Esta unidad introduce conceptos clave sobre cómo se estructura y desarrolla el software web, incluyendo tecnologías frontend y backend.

Objetivos de la Unidad

  • Que los estudiantes repasen conceptos básicos de desarrollo web que han aprendido durante su formación académica y que son indispensables para los siguientes módulos del Programa Células Plenum.
  • Que los estudiantes profundicen en tópicos intermedios y avanzados de programación con JavaScript.

Contenido

3.1 HTTP

El protocolo HTTP (Hypertext Transfer Protocol) es el protocolo fundamental para la comunicación entre navegadores web y servidores. Es un protocolo de tipo request-response, donde el cliente (navegador) envía una solicitud al servidor y este responde con una respuesta.
